# DingIt Working Rules

## Open Created Artifacts

When a Jira ticket, repository, pull request, or markdown document is created or moved for this project, open it immediately for review.

Current opening targets:

- Jira tickets: browser
- Repositories and pull requests: browser
- Markdown documents: local markdown viewer, preferably MdFilePad when available

If the preferred viewer is not available on PATH, locate it first. If it cannot be found quickly, use the system default markdown association and report that fallback.
